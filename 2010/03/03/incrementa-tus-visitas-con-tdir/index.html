Incrementa Tus Visitas Con TDir

Mar 3rd, 2010 | Comments

Ahora les presento este proyecto personal. TDir es un directorio Web mediante el cual puedes hacer tu Web cada vez más conocida en internet. Tu Web va ganando importancia con el tiempo en la medida en que va siendo mencionada por otras Webs, esto está medido por el pageRank, cosa que TDir incluye para medir cuan popular se ha vuelto o se sigue volviendo con el tiempo tu página Web.

Ventajas gratuitas:
  • PageRank de Google para cada Web.
  • Contador de clicks; para que sepas cuantas personas han visto tu Web.
  • Sistema de votación.
  • Inclusión en buscadores.
  • Posibilidad de tener tu Web siempre en la página principal.
Estas son las más importantes por el momento, pero el equipo de desarrollo seguirá trabajando por nuevas actualizaciones y mejoras en el servicio. Incluye tu Web ya mismo totalmente gratis y empieza a ganar visitas y a aumentar la importancia de Web en la red. Visitar: http://tdir.thxou.com
