[Video] Test De Velocidad Entre Safari Mobile Y Windows Mobile 7

Jul 30th, 2010 | Comments

Aquí os traigo otra comparativa y test de velocidad. Esta vez entre 2 navegadores móvil de 2 dispositivos móvil muy actuales: Safari Mobile en iPhone y el navegador de un Windows Mobile 7. Se nota la diferencia que marca el iPhone sobre otros móviles. Safari carga mucho más rápido las páginas que el Windows Mobile 7, aunque también hay que considerar el hardware y el rendimiento del OS de cada uno. Aquí les dejo el video:
Aunque Microsoft dijera que el Windows Mobile 7 era lo nunca antes visto, esta claro que se refería a los que aún no han probado el iPhone.
