[Mac] Simulador De N64 Para Mac

Jul 19th, 2010 | Comments

Yo soy uno de los que le gustaba el Nintendo 64 alla por el año 2000, pero lo dejé de jugar desde entonces y ahora me encuentro con esto: Un emulador de N64, nada menos. Para lo que son fans de esta consola y nunca encontraron una nativa para Mac OS X, aquí les dejo esta, se llama SixtyForce y va muy bien, incluso puedes jugar en modo de pantalla completa, modo ventana, utilizar un teclado y hasta un gamePad para jugar.

Las ROM’s (Los juegos) no vienen con el simulador, así que tienes que descargarlos manualmente para poder jugar. Puedes encontrar algunas aquí:
http://www.coolrom.com/roms/n64/ http://www.doperoms.com/roms/Nintendo_64.html
Cuando descargues las ROMs te vendrán comprimidas en archivos .zip, así que tendrás que descomprimirlas haciendo doble clic sobre ellos. Al hacer esto verás un fichero con extensión .z64, este es el formato soportado por el simulador.

Para jugar este juego solo tienes que hacer otra vez doble clic sobre el juego con extensión .z64 y aceptar las ventanas de alerta que aparezcan y listo. Utiliza la barra de menús para utilizar la pantalla completa: Options > Run Full Screen. Descargar el simulador para N64.
